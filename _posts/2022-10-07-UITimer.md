## Timer 사용법

타이머 설정하기

```swift
var timer = Timer()

override func viewDidLoad() {
    self.timer = Timer.scheduledTimer(withTimeInterval: 1, repeats: true, block: { in
        updateCounting()
    })
}

func updateCounting(){
    print("counting...")
}
```
