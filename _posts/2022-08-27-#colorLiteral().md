# #colorLiteral()

기존의 색을 설정하는 방법은 UIColor와 colorLiteral이 있었습니다.
하지만 최근 업데이트로 인해 #colorLiteral()에 대해 간략하게 설명해보려 합니다.

# 사용 방법

사용 방법은 굉장히 쉽습니다.

```swift
mainLabel.background = #colorLiteral()
```

이와 같이 원하는 값을 <mark>#colorLiteral</mark>를 이용해 넣어주기만 하면 됩니다.
